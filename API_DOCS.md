# API Documentation

## TechFront 2024 Event API

This document outlines the configuration and data structure for the TechFront 2024 Conference.

### Event Configuration

All event settings are stored in `config.json` with the following structure:

```json
{
  "event": {
    "name": "TechFront 2024",
    "date": "2024-11-15 to 2024-11-16",
    "location": "Mumbai Convention Center"
  }
}
```
